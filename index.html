Global Environmental Data Visualizations

Project by Meysam Jamali

About This Project

This project visualizes global environmental data to increase awareness and encourage informed actions. We utilize data from sources like NASA, World Bank, UN Statistics Division, and Our World in Data to create interactive charts highlighting key environmental trends.

Data Sources: World Bank, UN Statistics, Our World in Data, NASA Climate
